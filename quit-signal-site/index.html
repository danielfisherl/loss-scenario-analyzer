How To Read It

How to think about quitting

If runway is strong, the result may support leaving or taking time off. If runway is short, the answer is usually to reduce spend or wait for a payout.

The goal is not certainty. It is to make the tradeoffs visible enough that you can choose without hand-waving.

Quick FAQ

Is this financial advice?

No. It is a structured way to think through readiness, runway, and tradeoffs.

What if I’m burned out but my runway is okay?

That usually shifts the decision toward timing, not just savings.

What if I have a bonus or vest coming soon?

Run both scenarios. Waiting can matter, but only if the payout changes the decision.
