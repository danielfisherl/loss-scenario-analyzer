# QuitSignal static site

This folder is a standalone static website ready for GitHub Pages.

## Files
- `index.html` homepage
- `calculator.html` quit-job calculator
- `how-much-savings-to-quit-tech-job.html` article page
- `quit-before-rsu-vesting.html` article page
- `assets/styles.css` shared styles
- `assets/calculator.js` calculator logic
- `sitemap.xml` search indexing map (update domain placeholders)
- `robots.txt` crawler rules (update domain placeholders)

## Deploy to GitHub Pages
1. Create a new public GitHub repository.
2. Upload **the contents** of this folder (not the folder itself) to repository root.
3. In GitHub: `Settings -> Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`, then save.
6. Wait for Pages to build, then open your URL.

## Important SEO step
Update `YOUR-USERNAME` and `YOUR-REPO-NAME` in:
- `robots.txt`
- `sitemap.xml`

Then commit again.
